SeatWise Project Specifications

Logic & Algorithms

Room Allocation

Greedy Sequential Approach: Sorts students by entry order and fills rooms to capacity. Implements basic capacity constraint validation before assignment.

Matrix Generation

2D Grid Mapping: Maps student lists into a coordinate system (Row, Col) based on room dimensions. Future logic will implement neighbor-check constraints.
