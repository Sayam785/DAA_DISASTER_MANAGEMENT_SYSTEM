🔺 Heap Priority Queue — Disaster Ordering

Shows active disasters sorted by the max-heap. Priority: Emergency flag → Severity → Timestamp. Push O(log n), Pop O(log n).

📍 Dijkstra's Shortest Path

Finds shortest road route from a volunteer's location to a disaster site using Dijkstra on the Dehradun city graph. O((V+E) log V).

🎒 0/1 Knapsack DP — Resource Optimization

Selects optimal volunteer subset under a resource budget using bottom-up Dynamic Programming. O(n × W) time and space.

🔗 Disjoint Set / Union-Find — Network Connectivity
