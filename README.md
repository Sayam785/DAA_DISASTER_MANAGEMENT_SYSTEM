# ResQFlow
DAA Based Intelligent Disaster Response and Resource Optimization System

## Algorithms Implemented
- Graph Representation (Adjacency List)
- Dijkstra Shortest Path Algorithm
- Priority Queue (Heap)
- Greedy Algorithm for Volunteer Allocation
- Dynamic Programming for Resource Optimization
- Disjoint Set (Union-Find) for Connectivity

## Project Description
This system models a city as a weighted graph and optimizes disaster response by prioritizing disasters, computing shortest routes, allocating volunteers efficiently, and ensuring connectivity using classical Design and Analysis of Algorithms.
